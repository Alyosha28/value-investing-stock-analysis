# 快速开始指南

## 🚀 立即运行

### 1. 分析一只股票
```bash
python main.py 600519
```

### 2. 查看结果
报告将自动生成在 `output/` 目录：
- `sh.600519_贵州茅台_分析报告.txt` - 文本分析报告
- `sh.600519_贵州茅台_技术分析.png` - 技术图表

## 📝 基本命令

| 命令 | 说明 |
|------|------|
| `python main.py 600519` | 分析贵州茅台 |
| `python main.py 000001` | 分析平安银行 |
| `python main.py 600519 000001` | 同时分析两只股票 |
| `python main.py --help` | 查看帮助 |

## 🎯 支持的股票

- **上海A股**：600xxx（如 600519 贵州茅台）
- **深圳A股**：000xxx（如 000001 平安银行）
- **创业板**：300xxx
- **科创板**：688xxx

## 📊 分析内容

1. **股票基本信息**
   - 代码、名称、行业

2. **格雷厄姆分析**
   - PE、PB、ROE
   - 安全边际评分
   - 投资建议

3. **巴菲特分析**
   - 自由现金流
   - 股息率
   - 经济护城河
   - 内在价值评估

4. **技术分析**
   - MA（移动平均）
   - RSI（相对强弱）
   - 布林带
   - MACD
   - 买卖信号

5. **AI 智能分析**
   - 市场解读
   - 风险提示
   - 投资建议

## ⚡ 快速示例

```bash
# 1. 分析茅台
python main.py 600519

# 2. 分析苹果概念股
python main.py 000333
```

## 🔍 查看报告

Windows:
```bash
notepad output\sh.600519_贵州茅台_分析报告.txt
```

或直接打开 `output` 文件夹查看所有报告。

## 💡 常见问题

### Q: 程序运行很慢？
A: 首次运行需要下载数据，后续会使用缓存。

### Q: 某些字段显示 N/A？
A: baostock 的财务数据有限，这是正常现象。

### Q: 如何获取更完整的数据？
A: 注册 Tushare 获取 API token，详见 数据源配置指南.md

### Q: 生成的图表在哪里？
A: 在 `output` 文件夹，文件名包含"技术分析.png"
