```bash
# 查看报告列表
Get-ChildItem output\*.txt

# 查看图表
Get-ChildItem output\*.png

# 查看最新报告
Get-ChildItem output\*.txt | Sort-Object LastWriteTime -Descending | Select-Object -First 1
```

## 💡 提示

1. **首次运行较慢** - 需要下载数据
2. **查看日志** - 使用 `--verbose` 参数
3. **批量分析** - 使用空格分隔股票代码
4. **报告位置** - 在 `output/` 文件夹

## 🆘 常见问题

### Q: 程序报错？
A: 查看日志输出，常见问题：
- 数据源连接失败 → 重试即可
- 格式化错误 → 已修复
- 缺少依赖 → `pip install -r requirements.txt`

### Q: 数据不完整？
A: 这是正常现象。baostock 提供：
- ✅ 历史行情（完整）
- ✅ ROE等财务指标
- ⚠️ PE/PB等估值（不支持）

### Q: 如何获取完整数据？
A: 注册 Tushare（https://tushare.pro）
