# A股数据源配置指南

## 当前问题诊断

akshare 连接失败可能的原因：
1. **服务器限流**：访问频率过高被限制
2. **网络问题**：网络连接不稳定
3. **IP限制**：某些数据源对IP有限制
4. **数据源维护**：数据源正在维护

## 解决方案

### 方案一：使用 Baostock（推荐 - 免费稳定）

#### 1. 安装 baostock
```bash
pip install baostock
```

#### 2. 修改代码使用 baostock
在 `data_fetcher.py` 中集成 baostock 作为备选：

```python
from data_fetcher_backup import BackupDataFetcher

# 在主程序中
backup_fetcher = BackupDataFetcher()
if backup_fetcher._baostock:
    # 使用 baostock 获取数据
    stock_info = backup_fetcher.get_stock_info_baostock('sh.600519')
    historical = backup_fetcher.get_historical_data_baostock('sh.600519')
else:
    # 回退到 akshare
    print("Baostock 不可用，请检查安装")
```

### 方案二：使用 Tushare（数据丰富 - 需要注册）

#### 1. 注册 Tushare 账号
访问 https://tushare.pro/register 注册

#### 2. 安装 tushare
```bash
pip install tushare
```

#### 3. 使用示例
```python
import tushare as ts

# 设置 token（在 Tushare 网站获取）
ts.set_token('your_token_here')
pro = ts.pro_api()

# 获取股票数据
df = pro.daily(ts_code='600519.SH', start_date='20230101', end_date='20240101')
```

### 方案三：使用 EFinance（轻量级）

#### 1. 安装
```bash
pip install efinance
```

#### 2. 使用示例
```python
import efinance as ef

# 获取股票数据
stock_data = ef.stock.get_quote_history('600519')
print(stock_data)
```

### 方案四：调整 akshare 使用策略

#### 1. 使用代理（如有）
```python
import os
os.environ['HTTP_PROXY'] = 'http://your_proxy:port'
os.environ['HTTPS_PROXY'] = 'http://your_proxy:port'
```

#### 2. 错峰使用


## 快速修复：运行备选方案

运行以下命令测试 baostock：
```bash
python data_fetcher_backup.py
```

如果成功，你可以在主程序中添加以下逻辑：

```python
# 在 main.py 或 data_fetcher.py 中
try:
    # 尝试 akshare
    data = fetcher.get_stock_data(stock_code)
    if not data:
        raise Exception("akshare failed")
except:
    # 回退到 baostock
    backup_fetcher = BackupDataFetcher()
    data = backup_fetcher.get_all_data(stock_code)
    backup_fetcher.close()
```

## 数据源对比

| 数据源 | 免费 | 数据丰富度 | 稳定性 | 安装难度 |
|--------|------|-----------|--------|---------|
| akshare | ✓ | ★★★★★ | ★★★☆☆ | 简单 |
| baostock | ✓ | ★★★★☆ | ★★★★★ | 简单 |
| tushare | 部分免费 | ★★★★★ | ★★★★☆ | 中等 |
| efinance | ✓ | ★★★☆☆ | ★★★★☆ | 简单 |

## 建议

1. **开发测试**：使用 baostock（最稳定）
2. **生产环境**：组合使用多个数据源，互为备份
3. **数据完整性**：优先使用 tushare（需要注册）
4. **快速原型**：继续使用 akshare（最方便）

## 配置文件示例

创建 `config.py`：

```python
# 数据源配置
DATA_SOURCE = {
    'primary': 'akshare',      # 主数据源
    'backup': 'baostock',      # 备用数据源
    'retry_times': 3,          # 重试次数
    'timeout': 30,             # 超时时间（秒）
    'delay': 2                 # 请求间隔（秒）
}

# API Keys（如需要）
TUSHARE_TOKEN = 'your_token'  # Tushare token
```

## 监控和日志

建议添加数据源健康检查：

```python
def check_data_source_health():
    """检查数据源可用性"""
    sources = {
        'akshare': test_akshare,
        'baostock': test_baostock,
        'tushare': test_tushare
    }
    
    results = {}
    for name, test_func in sources.items():
        try:
            results[name] = test_func()
        except:
            results[name] = False
    
    return results
```
