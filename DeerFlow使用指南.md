# DeerFlow 使用指南

## DeerFlow 是什么？

DeerFlow（**D**eep **E**xploration and **E**fficient **R**esearch **Flow**）是一个开源的**超级智能体框架**，可以：
- 🤖 编排多个子智能体协同工作
- 💾 长期记忆和上下文管理
- 🐳 沙箱安全执行环境
- 🔧 可扩展的技能系统

## 快速启动

### 1. 启动服务

```bash
cd d:\自做ai股票分析\deer-flow
make dev
```

### 2. 访问界面

打开浏览器访问：**http://localhost:2026**

## 使用方法

### 基本对话

1. 在界面顶部的输入框中输入您的问题
2. 按 Enter 或点击发送按钮
3. 等待AI分析并返回结果

### 使用场景示例

#### 1. 股票分析
```
请帮我分析一下贵州茅台（600519）这只股票的投资价值
```

DeerFlow会：
- 自动联网搜索最新信息
- 执行代码获取财务数据
- 分析技术指标
- 生成综合报告

#### 2. 代码编写
```
用Python写一个股票技术指标计算的脚本
```

#### 3. 数据分析
```
分析最近一年的A股市场趋势
```

#### 4. 研究报告
```
生成一份关于新能源汽车行业的研究报告
```

### 高级功能

#### 子智能体协作
DeerFlow会自动将复杂任务分解给多个子智能体处理：
- 研究子智能体：搜索和分析信息
- 编码子智能体：执行代码和计算
- 写作子智能体：生成报告和总结

#### 沙箱执行
所有代码都在隔离的沙箱环境中执行，保证安全：
- 网络访问受限
- 文件系统隔离
- 无法访问本地敏感文件

#### 长期记忆
- 跨会话记住重要上下文
- 自动总结和索引历史对话

## 与股票分析系统的结合

您的股票分析系统可以与DeerFlow结合使用：

1. **DeerFlow负责**：
   - 自然语言交互
   - 联网搜索最新信息
   - 智能分析和解读
   - 生成自然语言报告

2. **股票分析系统负责**：
   - 结构化数据获取（PE、PB、ROE等）
   - 技术指标计算
   - 格雷厄姆/巴菲特模型分析
   - 批量股票分析

### 结合使用示例

```
在DeerFlow中输入：
"请分析600519贵州茅台，并结合本地股票分析系统生成完整投资报告"
```

DeerFlow会调用本地股票分析脚本，获取数据后整合分析。

## 常见问题

### 1. 启动失败
- 检查nginx是否运行：`nginx -v`
- 检查端口2026是否被占用
- 查看日志排查错误

### 2. API调用失败
- 检查网络连接
- 确认API密钥有效
- 查看API配额是否用完

### 3. 代码执行超时
- 复杂计算可能需要更长时间
- 尝试简化查询范围
